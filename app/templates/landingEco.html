Ecosistema corporativo

El ecosistema tiene como objetivo la búsqueda de soluciones a nivel corporativo. De esta forma busca unir la brecha entre usuarios operativos, coordinación y toma de decisiones corporativa.

Características de valor

Operatividad

Tareas conjuntas

A través de la plataforma se puede coordinar, ejecutar y compartir resultados. También es posible coordinar proyectos de equipos distribuidos geográficamente.

Reportabilidad

Es posible compartir resultados a jefaturas directas a través de la plataforma.

Toma de decisiones

Gráficos de avance

Obtención de gráficos de avance a partir de un conjunto de tareas a partir de dimensiones de datos requerida. Por ejemplo, gráfico de tonelaje contra ejecución de cierta tarea ejecutada en el tiempo con datos actualizados.

Inteligencia artificial

LObtención de metadata a partir de análisis del trabajo en la plataforma por su empresa. Aplicación de técnicas de minería de datos, machine learning y otros.

Productos principales

El ecosistema de tareas comprende una serie de productos. Cada uno de ellos brindan una mirada complementaria en busca de brindar un servicio coorporativo de ejecución, administración y apoyo a la toma de decisiones.

Tasker

  • Ejecución de comandos y tareas en la nube
  • Aumento de productividad operativa
  • Colaboración entre usuarios

Orchest

  • Automatiza tareas colaborativas
  • Posibilita trabajos multidisciplinarios
  • Mejora en productividad colaborativa

Commander

  • Completo monitoreo de tareas y workflows
  • Análisis de data generada
  • Apoyo en toma de decisiones

Productos complementarios

Con el fin de brindar apoyo a nuevos usos de las funcionalidades de este ecosistema se definen una serie de productos complementarios.

Rest API

  • Acceso a ejecución de tareas
  • Acceso por interfaz web simple y disponible
